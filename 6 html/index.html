Welcome to Windows 7 media website

Windows 7 was released on 22 October 2009. Under the slogan "Your PC, simplified", the operating system introduced many changes and overhauls compared to its predecessor, Windows Vista. While the Aero interface was largely retained, it was considerably revamped, with the most notable feature being the new enlarged Taskbar that introduced the ability to pin the applications directly to the bar instead of previously-available Quick Launch Bar.

Another important overhaul was the updated Windows Explorer, which attained a feature called Libraries. These are collections of different folders with similar content. Four libraries were present after the installation - Documents, Music, Pictures and Videos. ALso, Windows 7 introduced HomeGroup, an ad hoc network connection with other Windows 7 devices in the local home network. It has the ability to share folders between different computers in the network with ease and without any Internet connection.
